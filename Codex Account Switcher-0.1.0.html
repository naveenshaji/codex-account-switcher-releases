Codex Account Switcher 0.1.0

Initial public desktop release.

Included

  • Multi-account ChatGPT OAuth management for Codex desktop and CLI
  • One-click active account switching
  • Per-account usage remaining visualization for 5h and weekly windows
  • Local history tracking with forecast bars and reset detection
  • First-run onboarding in the menubar panel
  • Sparkle auto-update support for packaged builds

Notes

  • Running Codex desktop sessions still require restart after account switches
  • CLI sessions still require opening a new session after account switches
